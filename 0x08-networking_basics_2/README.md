# Networking basics 1
The project it aims to learn about what is localhost/127.0.0.1, what is 0.0.0.0, what is `/etc/hosts` and how to display the machine's active network interfaces.

## Technologies
+ Scripts written in Bash 5.1.8(1)
+ Tested on Ubuntu 20.04 LTS

## Files
The files contained in this repository are:

| Filename | Description |
| -------- | ----------- |
| `0-change_your_home_IP` | Bash script that configures an Ubuntu server |
| `1-show_attached_IPs` | Bash script that displays all active IPv4 IPs on the machine |
| `100-port_listening_on_localhost` | Bash script that listens on port `98` on `localhost` |
